# Expense Tracker

## Project Overview

Expense Tracker is a web-based application built using HTML, Tailwind CSS, and JavaScript that helps users manage their daily expenses. The application allows users to add, view, filter, and delete expenses while automatically calculating the total amount spent. Expense data is stored in Local Storage, ensuring persistence even after the browser is refreshed.

This project demonstrates the use of Object-Oriented Programming (OOP), Functional Programming concepts, ES6 features, Local Storage, and DOM Manipulation.

---

## Features

### Expense Management

* Add new expenses with:

  * Description
  * Amount
  * Category
* View all recorded expenses.
* Delete expenses individually.
* Filter expenses by category.
* Display the total amount spent.
* Display the total number of expenses.

## Usage

### Adding an Expense

1. Enter the expense description.
2. Enter the amount.
3. Select a category.
4. Click **Add Expense**.

### Filtering Expenses

1. Select a category from the filter dropdown.
2. The application displays only matching expenses.

### Deleting an Expense

1. Click the **Delete** button next to an expense.
2. The expense is removed immediately.
